# AI Literacy Coach 🧠✨

An interactive educational platform designed to help students master the art of prompting AI effectively, responsibly, and ethically. Aligned with **UN Sustainable Development Goal #4 (Quality Education)**.

## 🚀 Overview

The AI Literacy Coach doesn't just give you answers—it teaches you how to **think** with AI. By analyzing your prompts across several metrics (Clarity, Specificity, Depth, and Professionalism), the coach helps you bridge the AI literacy gap using the **S.P.E.C.I.F.I.C** framework.

## ✨ Features

- **Interactive Prompt Lab**: Practice your prompting skills with real-time feedback.
- **Deep Analysis**: Powered by Google Gemini, get a score and detailed breakdown of your prompt quality.
- **Educational Frameworks**: Learn the S.P.E.C.I.F.I.C method (Situation, Purpose, Expectation, Constraints, Information, Format, Importance, Context).
- **Premium Interface**: A modern, glassmorphic dark-mode design with smooth animations.
- **Privacy First**: Your API key is stored locally and never sent to our servers.

## 🛠️ Setup & Deployment

### Local Development

1. Clone the repository.
2. Open `index.html` in any modern browser.
3. Enter your [Google Gemini API Key](https://aistudio.google.com/app/apikey) in the settings.

### Deployment (Vercel)

This project is ready for one-click deployment to Vercel.

1. Push this repository to GitHub.
2. Connect the repository to [Vercel](https://vercel.com).
3. Deploy!

## 🌍 SDG Alignment

This project directly addresses **SDG #4** by providing equitable access to AI literacy tools, ensuring that youth from all backgrounds can develop the skills needed for the modern workforce.
